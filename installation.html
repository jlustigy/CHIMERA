Installation¶

Install with Pip¶

pip install chimera

Install with Git¶

git clone https://github.com/mrline/chimera.git
cd chimera
python setup.py install

Download the Reference Files¶

Download all the reference data in this DROP BOX HERE.

Download all of the *.h5 files and the subfolders CHEM and MIE_COEFFS into the ABSCOEFF_CK folder on your computer. It may take a minute…. Once that is done, we will ask you for the path to the ABSCOEFF_CK folder.
